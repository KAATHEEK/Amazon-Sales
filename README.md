# 📦 Amazon Sales Analysis

A comprehensive **Exploratory Data Analysis (EDA)** project focused on uncovering patterns, trends, and actionable insights from Amazon sales data.

---

## 📌 Project Overview

This project explores an Amazon sales dataset through structured data analysis — from cleaning and preparation through to visual storytelling. The goal is to identify what drives sales performance and surface meaningful patterns across product categories, regions, and time periods.

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|---|---|
| **Python** | Core programming language |
| **Pandas** | Data manipulation & cleaning |
| **NumPy** | Numerical computations |
| **Matplotlib** | Static visualizations |
| **Seaborn** | Statistical data visualization |
| **Jupyter Notebook** | Interactive analysis environment |

---

## 🔍 Key Analysis Areas

- **Data Cleaning** — Handling missing values, duplicates, and data type corrections
- **Data Exploration** — Understanding the structure, distributions, and summary statistics of the dataset
- **Exploratory Data Analysis (EDA)** — Identifying trends, correlations, and anomalies
- **Data Visualization** — Charts and plots to communicate insights clearly

---

## 📊 Key Questions Explored

- What are the top-performing product categories by revenue?
- Are there seasonal or monthly sales trends?
- Which regions or fulfillment channels drive the most sales?
- How do ratings and reviews relate to sales performance?
- What factors contribute to order cancellations or returns?
